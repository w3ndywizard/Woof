Woof

Custom Model for Codeigniter - Less Easy Flexible

View project onGitHub

What's Woof?

Woof is custom model for Codeigniter which's provides comfortable methods such as CRUD. It'll take an easy to use, short and absolutely it's flexible.

Installation

Copy file Woof.php to libraries folder and set auto load library in config/autoload.php

$autoload[‘libraries’] = array(‘woof’);

Implement your Model

Every models in your application must extends from Woof class. Afterward you must define attributes in attributeRule() and set database, table name and other in initialize() as example.

class foo extends Woof
{
  protected function attributeRule()
  {
    return array(
      // Define id field and set Primary key
      array('name'=>'id', 'pk'=>true),
      array('name'=>'fname'),
      array('name'=>'sname')
    );
  }
  protected function initialize()
  {
    // Set select database
    $this->db = 'default'
    // Set table name
    $this->db = 'foo'
  }
}

Note: pk is sign to indicate that attribute is primary key. It'll combine with another method such as findByPk(), updateByPk(), deleteByPk() if you not set those method will not work. By the way you can set pk attribute through pkField attribute in initialize().

CRUD

CRUD is basic function that interacting with database, It contains four functions such as CREATE, READ, UPDATE and DELETE.

insert()

Add single record to database. You can set value to attributes which you have defined in attributeRule() method.

Syntax

bool insert();

Usage

$this->load->model('foo');
$this->foo->fname = 'Petit';
$this->foo->sname = 'Wong';
$this->foo->insert();

set()

Alternative way to setting value through this method rather than put value directly to attribute

Syntax

void set(fieldname, value);

Usage

$this->load->model(‘foo’);
$this->foo->set(‘fname’, ‘Petit’);

update()

Update records by specific conditions. You can pass Array or Criteria object perform conditions. See also How to define conditions.

Syntax

bool update(Mixed);

Usage

$this->load->model(‘foo’);
$this->foo->fname = ‘Windy’;
$this->foo->update(array(‘fname’=>’Petit’));

updateByPk()

Change data in record by specific primary key value

Syntax

bool updateByPk(value);

Usage

$this->load->model(‘foo’);
$this->foo->fname = ‘Windy’;
$this->foo->updateByPk(1);

delete()

Remove records by specific conditions. You can pass Array or Criteria object perform conditions. See also How to define conditions.

Syntax

bool delete(mixed $conditions);

Usage

$this->load->model(‘foo’);
$this->foo->delete(array(‘fname’=>’Petit’));

deleteByPk()

Remove record by specific primary key value

Syntax

bool deleteByPk(mixed $pkvalue);

Usage

$this->load->model(‘foo’);
$this->foo->deleteByPk(1);

find()

Get single record by specific conditions. If result not found it’ll return blank object. You can pass Array or Criteria object perform conditions. See also How to define conditions.

Syntax

object find([mixed $conditions]  [, array $fieldsname]);

Usage

$this->load->model(‘foo’);
$row = $this->foo->find(array(‘id’=>1));
echo $row->fname . ‘ ‘ . $row->sname; // Petit Wong

// You can also specific fields name
$row = $this->foo->find(array(‘id=>1’), array(‘fname’));
echo $row->fname . ‘ ‘ . $row->sname; // Petit

// Pass with string
$row = $this->foo->find('id = 1', 'fname');
echo $row->fname;

findAll()

Get records by specific conditions. You can pass Array or Criteria object perform conditions. See also How to define conditions.

Syntax

array findAll([mixed $conditions] [, array $fieldsname]);

Usage

$this->load->model(‘product’);

// Retrieve all records
$this->product->findAll(); 

// Retrieve records that has type_id = 1
$this->product->findAll(array(‘type_id’ => 1)); 

// Retrieve all records and select only product_id and productname fields
$this->product->findAll(null, array(‘product_id’,’product_name’));

findByPk()

Get a record by specific primary key value

Syntax

object findByPk(mixed $value [, array $fieldsname]);

Usage

$this->load->model(‘foo’);
$row = $this->foo->findByPk(1)
echo $row->fname . ‘ ‘ . $row->sname; // Petit Wong

// Put fields parameter
$row = $this->foo->findByPk(1, array(‘fname’);
echo $row->fname . ‘ ‘ . $row->sname // Petit

findBySql()

Get record using SQL query command;

Note: Return type is CI_DB_mysql_result, you can access result by using result(), row() etc.. that refer on Codeigniter' result and This method not support FK Binding feature.

Syntax

array findBySql(string $sqlcommand);

Usage

$this->load->model(‘foo’);
$this->foo->findBySql(‘SELECT * FROM foo’);

How to define command

You can direct access to db object which's Active Record object of Codeigniter and add command to query. By the way, there are several ways to assign command or condition by using Array or Criteria that hold your command. And now I'll show three ways how to assign command to query data.

1) Direct through db object

The db is Active Record object’s Codeigniter. You can access through db() method to assign command.

$this->load->model(‘foo’);
$this->foo->db->select(‘*’);
$this->foo->db->where(‘field1 = 1’);

Note: If you notice, Nearly all of commands refer to Codeigniter's Active Record object. You can reference object in this source.

2) Use Array

You can assign command by using multiple array. The commands contain many of arrays which has different key that represent command to query data.

$criteria = array = (
          array(‘select’ => ‘*’),
          array(‘where’ => ‘field1 = 1’)
            );

3) Use Criteria

Criteria is similar Array, but Criteria is object.

Criteria is similar Array, but Criteria is object.
$obj = new Criteria
$obj->select = ‘*’;
$obj->where = ‘field1 = 1’;

List of command

select

Select field members.

Parameter format

$obj = new Criteria;
$obj->select = 'field1, field2, field3';
$obj->select = array('field1, 'field2', 'field3');

distinct

Distinct value.

Parameter format

$obj = new Criteria;
$obj->distinct = true; // default false

like

Search value by keyword.

Parameter format

$obj = new Criteria;
$obj->like = 'field1 : %value%';
$obj->like = 'field1 : value : before'; // use before, after or both

or_like

Search value by keyword and add "OR" operator.

Parameter format

$obj = new Criteria;
$obj->or_like = 'field1 : %value%';
$obj->or_like = 'field1 : value : before'; // use before, after or both

not_like

Search value not in specific keyword.

Parameter format

$obj = new Criteria;
$obj->not_like = 'field1 : %value%';
$obj->not_like = 'field1 : value : before'; // use before, after or both

or_not_like

Search value not in specific keyword and add "OR" operator.

Parameter format

$obj = new Criteria;
$obj->or_not_like = 'field1 : %value%';
$obj->or_not_like = 'field1 : value : before'; // use before, after or both

join

Join table.

Parameter format

$obj = new Criteria;
$obj->join = 'table2 : table2.id = table1.id';

// use join optional e.g. left, right, outer, inner, left outer and right outer
$obj->join = 'table : table2.id = table1.id : left';

left

Left join table.

Parameter format

$obj = new Criteria;
$obj->left = 'table2 : table2.id = table1.id';

right

Right join table.

Parameter format

$obj = new Criteria;
$obj->right = 'table2 : table2.id = table1.id';

outer

Outer join table.

Parameter format

$obj = new Criteria;
$obj->outer = 'table2 : table2.id = table1.id';

inner

Inner join table.

Parameter format

$obj = new Criteria;
$obj->inner = 'table2 : table2.id = table1.id';

left_outer

Left outer join table.

Parameter format

$obj = new Criteria;
$obj->left_outer = 'table2 : table2.id = table1.id';

right_outer

Right outer join table.

Parameter format

$obj = new Criteria;
$obj->right_outer = 'table2 : table2.id = table1.id';

where

Set where conditions.

$criteria = array(
  array('where' => 'field1 = value')
);

// Add multiple
$criteria = array(
  array('where' => 'field1 = value1'),
  array('where' => 'field2 != value2')
);

or_where

Set where conditions and add "OR" operator.

$criteria = array(
  array('or_where' => 'field1 = value1')
);

where_in

Set IN condition.

// Array format
$criteria = array(
  array('where_in' => 'field : (value1, value2, value3)')
);

// Criteria format
$obj = new Criteria;
$obj->where_in = 'field : (value1, value2, value3)';

where_not_in

Set NOT IN condition.

$criteria = array(
  array('where_not_in' => 'field : (value1, value2, value3)')
);

or_where_not_in

Set NOT IN condition and add "OR" operator.

$criteria = array(
  array('or_where_not_in' => 'field : (value1, value2, value3)');
);

group_by

Group result by field.

$criteria = array(
  array('group_by' => 'field1');
);

// Multiple field
$criteria = array(
  array('group_by' => array('field1', 'field2'))
);

having

Set having command.

$criteria = array(
  array('having' => 'field : value')
);

// Set prevent escaping
$criteria = array(
  array('having' => 'field : value : true')
);

or_having

Set having command and add "OR" operator

$criteria = array(
  array('or_having' => 'field : value')
);

order_by

Order by specific field

$obj = new Criteria;
$obj->order_by = 'field ASC';

// Multiple fields
$obj = new Criteria;
$obj->order_by = 'field1 ASC, field2 DESC';

limit

Limit number of record.

$criteria = array(
  'limit' => 10
);

// Limit 20 rows and set start index at 10
$criteria = array(
  'limit' => '20, 10'
);

offset

Set start index.

// Equality "limit => '10,5'"
$criteria = array(
  'limit' => 10,
  'offset' => 5
);

Binding Foreign Key

Woof provide function to retrieve record of Foreign key value. As image that show on below indicate about database relation between two tables.

You can retrieve record by this feature easily without create another function to get data.

Note: Don't support findBySql() method.

$this->load->model('person');

// Attribute "useDefaultResult" requires to set FALSE
$this->person->useDefaultResult = false;
$row = $this->person->findByPk(1);

// Print attributes
echo $row->id . '<br />';
echo $row->fname . ' ' . $row->sname . '<br />';
echo $row->state_id;

// Just add () to make attribute to method
echo $row->state_id()->state_id . ' ' . $row->state_id()->state_name; // 1 texas

// You can specific fields by pass string or array parameter.
echo $row->state_id('state_name')->state_name;
echo $row->state_id(array('state_name'))->state_name;

How to define?

You can define foreign key attribute in attributeRule() and set binding to related class as show in example

class person extends Woof
{
  protected function attributeRule()
  {
    return array(
      array('name'=>'id', 'pk'=>true),
      array('name'=>'fname'),
      array('name'=>'sname'),
      array('name'=>'state_id', 'fk'=>array(
                                'class'=>'state', // related class
                                'destAttr'=>'state_id' // related field
                               )                                            
    );
  }

  protected function initialize()
  {
    $this->db = 'default';
    $this->tableName = 'person';
  }
}

There's another way to bind foreign key by using bindFK() method. you can place in initailize().

Syntax

bindFK(string $sourceAttr, string $destAttr, string $class [, string $path]);
  • $sourceAttr - Attribute name that represent Foreign key in class.
  • $destAttr - Primary key name in related class
  • $class - Related class
  • $path - if model has sub folder, you can specify path here

If use this feature, You must always set attribute "useDefaultResult" to false is required after load any model.

$this->load->model('person');
$this->person->useDefaultResult = FALSE;
